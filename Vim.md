### Description
---
A powerful text editor that's built into most Linux Systems. 

### Code
---
##### Functions
---
```
i      # Enter insert mode
esc    # Exit 
:w     # Writethe file
:q     # Quit Vim
:wq    # Write Quit
:q!    # Quit without writing
```

---
```
vim <filename>
```
